1. Getting Tiled: here
  2. Setting up:
    • Go to Edit->Preferences... and change "Store tile layer data as:" to XML
    • Create new map: File->New...
    • Import tilesets (Map->Add External Tileset...) from files: Tiles.tsx, Teleports.tsx, Border.tsx in "tiled" directory
  3. Create your board. It should contain layer named "Map", on wich "Tiles" tileset should be used.
    It can contain also layer named "Border", wich "Border" tileset should be used. If certain tile is marked as "Border" it will not be used for spawning players or items.
    It is also possible to create one or more teleport layer (their names should begin with "Teleports", and tileset used should be "Teleports"). There should be at most one teleport of each kind on single teleport layer. The teleports will lead from layers with lower numbers to layers with higher numbers (Teleport from layer "Teleports1" will lead to teleport on "Teleports2" and so on). Teleport layer with highest number will lead to the layer with lowest number. If numbers are not used, the layers will be sorted lexicographically ("Teleportshaha", "Teleportslol", "Teleportswth" and so on).
  4. Save your board
  5. Run "tmx2brd.py" script from "sources" directory on saved file
  6. Put created *.brd file in "levels" directory
  7. Have fun playing on your newly created board
